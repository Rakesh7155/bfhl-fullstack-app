# 🚀 BFHL Full Stack Application

This project is a Full Stack application that processes flat edge relationships and converts them into hierarchical tree structures. It also detects invalid inputs, duplicate edges, and cycles in the graph.

---

## 📌 Features

- ✅ Parse edge relationships like `A->B`
- ✅ Build hierarchical tree structures
- ✅ Detect invalid inputs
- ✅ Handle duplicate edges
- ✅ Detect cycles in graph
- ✅ Calculate depth of tree
- ✅ Provide summary of results

---

## 🛠️ Tech Stack

- **Backend:** Node.js, Express.js
- **Frontend:** HTML, CSS, JavaScript
- **API Testing:** Postman / Browser
